Toggle Profile Information Visibility
######################################

.. tags:: learner, how-to
   
.. START TOGGLE PROFILE INFORMATION VISIBILITY

For any profile field, you can toggle the visibility of the profile information to other learners.

Options are: 

* **Just Me**: Only you can see this information.

* **Everyone on [Platform]**: Anyone who is a registered learner on the platform can see this information.

To toggle the visibility of the profile information, follow these steps:

#. Within the dashboard page, :ref:`choose a field to edit <Update your Profile Details>`.

#. Modify the **Who can see this** field by selecting the appropriate option.

#. Click the **Save** button.
