.. _Sync a Library update to your course:

Sync a Library update to your course
####################################

.. tags:: educator, how-to

.. admonition:: Publish Your Updated Library Content

    In order to sync a piece of content within a course to the latest version in
    the Library, the updated Library content must first be published before it
    will be available to sync within the course.

.. contents::
  :local:
  :depth: 1

.. _Sync within a unit:

Sync Library Unit Content
**************************

#. When a piece of content is updated and re-published in a Library, the sync
   button appears with the text :guilabel:`UPDATE AVAILABLE`.

   .. image:: /_images/educator_how_tos/library_sync_button.png
    :alt: A screenshot of a course content unit with a sync icon button in the top right corner

#. Click the button to start the sync process.

#. A modal pops up to show the old version and the new version of the content.
   Review the changes.

   .. image:: /_images/educator_how_tos/library_sync_review_changes.png
    :alt: A screenshot of the Preview Changes modal, which shows the difference between the old version and the new version of the content.

#. Choose to :guilabel:`Accept changes`, or :guilabel:`Ignore changes` if you'd
   prefer to keep your current version of the content.

.. _Sync subsections or sections:

Sync Library Subsections or Sections
************************************

From within a course, library content can be synced when a library is updated.

#. From a course section or subsection, click the "Sync" button.

   .. image:: /_images/educator_how_tos/library_sync_button.png
    :alt: A screenshot of a course content unit with a sync icon button in the top right corner

#. Alternatively, from a course page in Studio, select **Content > Library Updates** from the top menu bar.

   .. image:: /_images/educator_how_tos/library_updates_menu.png
      :alt: A course page in Studio, with the Content menu selected. The second item in the dropdown is the Library Updates option.

#. On the Content Libraries page, select **Review Content Updates**. If there is
   content to be updated, a notification dot will appear on this menu.

#. On the card for each Library with new content, there are options to
   :guilabel:`Review Updates`, :guilabel:`Ignore` the updates, or
   :guilabel:`Update` the library content in the course.

   .. image:: /_images/educator_how_tos/library_review_content_updates.png
      :alt: On the Library page, there are two tabs: Libraries (which show which libraries are used in the course) and Review Content Updates. In this screenshot, Review Content Updates is selected, and the notification dot indicates there is 1 change to sync.

#. Select :guilabel:`Review Updates` to preview the incoming library changes.
   There will be a preview of what content will be changing with the update: the
   left side of the preview shows whach changes to the library content will be
   made, and the left side shows what the result of those changes will look like
   in the course.

   .. image:: /_images/educator_how_tos/library_review_content_updates_2.png
      :alt: The left side panel says "Before" at the top, and the right side panel says "After" at the top.

#. Select :guilabel:`Accept changes` to make the changes to the library content
   in the course. Select :guilabel:`Ignore changes` to keep library content
   as-is in the course.
