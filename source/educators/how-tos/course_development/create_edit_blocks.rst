Create and Edit New Subsections and Sections
############################################

Libraries contain the same rich editing interfaces that are available in
courses. Subsections and sections can be edited and updated at any time.

.. note::

   Users must have **Library Author**, **Library Contributor**, or **Library Admin** permissions on a library to edit content.
   See :ref:`Add users to Libraries` for more details.

Create a New Subsection or Section
**********************************

#. Click on the :guilabel:`New` button at the top of the page. The Add Content sidebar will open.

#. Click on either the :guilabel:`Subsection` or :guilabel:`Section` button from the Add Content sidebar.

   .. image:: /_images/educator_how_tos/create_new_sub_or_section.png
      :alt: The Add Content right-hand sidebar on a Library page, with the Section and Subsection buttons highlighted

#. Add a title to your new Subsection or Section, and click :guilabel:`Create`. The new empty page will open.

Add New Content to a Subsection or Section
******************************************

#. From the new subsection or new section page, click the :guilabel:`+ Add Unit`
   button (for subsections) or the :guilabel:`+ Add Subsection` button (for
   sections) at the top of the page. The Add Content sidebar will open.

   .. figure:: /_images/educator_how_tos/add_unit_library_subsection.png
      :alt: The "+ Add Unit" button appears on the main Library page, at the top right of the page

      The Add Unit button appears within a Library Subsection
   
   .. figure:: /_images/educator_how_tos/add_subsection_library_section.png
      :alt: The "+ Add Subsection" button appears on the main Library page, at the top right of the page

      The Add Subsection button appears within a Library Section
   
   
#. Choose existing content, or create new content, to populate the subsections/sections.
